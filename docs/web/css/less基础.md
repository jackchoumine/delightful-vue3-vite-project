# LESS 基础

## 为何会有 less 

CSS 是一门非编程语言，没有变量、逻辑运算、运算等概念，随着 CSS 样式的扩展，会难以维护、难以扩展，产生大量冗余代码，所以产生了 less 、scss 等扩展语言，以解决 CSS 的缺点，这些扩展语言通过相应的预处理器（一种软件）处理，可转为 CSS。
